## 关于我

这里是你的个人介绍页。

你可以把这一页改成下面几种结构中的任意一种：

- 一段简短自我介绍
- 你的职业方向、研究兴趣或长期关注的话题
- 你的写作范围，比如技术、阅读、产品、生活记录
- 你常用的联系方式与社交平台

## 我在写什么

这部分适合回答“为什么要做这个博客”。

你可以写：

- 你希望长期记录什么
- 你想把这里做成知识库、作品集，还是生活博客
- 你希望读者在这里看到什么内容

## 我做过什么

如果你希望这个博客也承担作品展示功能，可以在这里放：

- 做过的项目
- 工作或学习经历
- 你比较满意的几项成果

## 联系方式

建议至少保留一种稳定联系方式：

- 邮箱
- GitHub
- 即时通讯
- 社交媒体主页
